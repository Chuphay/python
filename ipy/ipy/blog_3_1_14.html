What I want to do now, is try to fit this data to a model. The y-displacement is easy, it looks just like a bell curve, so let's try and fit this to normal distribution.

Next I want to see if I can model the time. This looks like a skewed normal distribution. After checking on stack overflow, this is what I got

In [7]: 
#http://stackoverflow.com/questions/5884768/skew-normal-distribution-in-scipy
#http://en.wikipedia.org/wiki/Skew_normal_distribution

from scipy.special import erf

def pdf(x):
    return 1/sqrt(2*pi) * exp(-x**2/2)

def cdf(x):
    return (1 + erf(x/sqrt(2))) / 2

def skew(x,e=0,w=1,a=0,p=1):
    t = (x-e) / p
    return 2 / w * pdf(t) * cdf(a*t)


df['time'].hist(bins = 100, histtype = 'step')
z,x=np.histogram(df['time'],bins=100)
x = x[:z.size]
fitpars, covmat = curve_fit(skew,x,z)
plot(x, skew(x, *fitpars), 'r-')
show()

And this seems like a nice fit as well.
